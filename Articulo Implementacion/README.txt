Para la utilizacion del codigo se utilizaron 2 libreria que manejan 
Transformadas Wavelets:

1 - La primera es PyWavelets, su pagina es https://pywavelets.readthedocs.io/en/latest/
y alli se encuentran diferentes instrucciones segun el sistema operativo
por lo general se utiliza el comando "pip install PyWavelets"

2 - La segunda es Yet Another Wavelet Toolbox o PyYAWT, http://pyyawt.readthedocs.io/
esta de la misma manera tiene instrucciones en la pagina pero comunmente se usa el comando
"pip install pyyawt"
